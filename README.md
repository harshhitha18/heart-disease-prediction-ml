# ❤️ Heart Disease Prediction System

## 🚀 Overview
Machine Learning-based system to predict heart disease risk using classification models and an interactive Streamlit interface.

## 🧠 Features
- Multiple ML models (Logistic, Random Forest, MLP)
- Hyperparameter tuning (GridSearchCV)
- Model evaluation (Accuracy, F1-score, ROC-AUC)
- Streamlit UI for real-time prediction
- CSV batch prediction support

## 🛠 Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy
- Streamlit

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
